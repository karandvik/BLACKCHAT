# BLACKCHAT

Если кому-то нужен маленький, удобный и шустрый web чат на PHP просто для общения без "украшений" то это то что нужно!

Чат создавался для очень медленных и нестабильных каналов, очень компактный и полнофунциональный!

Клиентская часть выполнена по технологии XHTML, CSS и JavaScript в клиенте
испльзуется менеджер задач, т.е. любые операции сначала помещаются в специальную очередь,
а потом выполняются, следующая задача выполняется если предыдущая завершилась успешно.
 
Серверная часть базируется на PHP, используются естественно сессии, данные хранятся в обычных файлах. 
Все разбито по функциям, в PHP скрипте имеются комментарии ко всем функциям.

Для дополнительной информации смотрите файл readme.txt в внутри архива
